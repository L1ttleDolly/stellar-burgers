# Stella Burger

Stella Burger — веб-приложение для заказа бургеров с возможностью регистрации, авторизации и персонализированного просмотра заказов.

## Архитектура

Приложение построено по принципу разделения ответственности:

- **Компоненты View** — отображение интерфейса.
- **Модель данных** — управление состоянием и структура данных.
- **Бизнес-логика** — полностью отделена от компонентов View.

Состояние приложения управляется с помощью **Redux Toolkit**.

Приложение покрыто тестами с использованием **Cypress** (энд-ту-энд) и **Jest** (юнит-тесты).

## Функционал

### Аутентификация
- Регистрация нового пользователя.
- Логин существующего пользователя.
- Восстановление пароля по электронной почте.
- Протектед маршруты для авторизованных пользователей.

### Конструктор бургеров
- Добавление ингредиентов в заказ.
- Оформление заказа доступно только авторизованным пользователям.

### Лента заказов
- Просмотр всех заказов с обновлением в реальном времени.
- Счетчик выполненных заказов за сегодня и за всё время.
- Просмотр статусов заказов («готовится», «выполнено») в модальном окне.

### Личный кабинет
- Просмотр своих заказов.
- Возможность редактировать логин, email и пароль.
- Выход из приложения.
